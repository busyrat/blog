# 通用快捷键

  • 切换 tab： cmd + option + 左右
  • 复制行：option + shift + 下
  • 关闭侧边栏 / 控制台：cmd + B / cmd + J
  • 创建分屏快捷键是 cmd + \ , 编辑器之间分屏切换快捷键是 cmd + 1 - 9
