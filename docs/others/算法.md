### 时间复杂度

可以理解为执行代码的行数

`O(1)` ：一次就可以取到目标元素；显而易见`O(n)`是需要 n 次；那么`O(log n)`是怎么来的呢？

假设有 n 个数，使用二分搜索：$n * (\cfrac{1}{2})^k = 1; k = \log_{2}n$

所以用 `log n`表达这种行为
