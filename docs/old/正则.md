## 总结

### 一些概念


| 元字符 | 元字符                            |
| :----- | :-------------------------------- |
| 基本   | `.` `\`  `|`                      |
| 字符组 | `[]` `[^]` `[-]`                  |
| 数量   | `{m,n}` `{m,}` `{m}` `+` `*` `?`               |
| 位置   | `^` `$` `\b` `\B` `(?=p)` `(?!p)` |
| 特殊   | `\d` `\D` `\s` `\S` `\w` `\W`     |
| 标志   | `g` `i` `m`                       |


简写

| 简写        | 表示                  | 记忆  |
| :---------- | :-------------------- | :---- |
| `\d` (`\D`) | `[0-9]`               | digit |
| `\w` (`\W`) | `[0-9a-zA-Z_]`          | word  |
| `\s` `\S`   | `[\t\v\n\r\f]`        | space |
| `.`         | `[^\n\r\u2028\u2029]` | all   |
| `?`         | `{0,1}`               |
| `+`         | `{1, }`               |
| `*`         | `{0,}`                |


#### 贪婪 和 惰性 （所有量词后面添加 `?` 都能实现惰性，并且管道符也是惰性的）

举例：

贪婪：能力范围内，越多越好 `\d{2,5}`

惰性：满足最低配就进入下次 `\d{2,5}?`

#### 位置的特性

- `/^hello$/` 写成 `/^^hello$$$/` 等效
